# Visualik Web

Portafolio web de Visualik, agencia de marketing digital.

## Estructura actual

- `index.html`: página principal en la raíz.
- `assets/css/styles.css`: hoja de estilos.
- `assets/js/main.js`: script principal.
- `assets/img/`: imagenes locales (ej. `favicon.png`).

## Desarrollo

1. Clonar el repositorio:

```bash
git clone https://github.com/EnzoGTorres/Visualik-web.git
cd Visualik-web
```

2. Abrir `index.html` en el navegador directamente o usar un servidor estático.

3. Modificaciones:

- CSS: `assets/css/styles.css`
- JS: `assets/js/main.js`

## Deploy

Subir al hosting de tu preferencia. Es un sitio estático, puede ir a Netlify, Vercel, GitHub Pages, etc.

## Notas

- La ruta de los assets está configurada con rutas relativas en `index.html`.
- Cualquier imagen local para el proyecto debe ir en `assets/img/`.
